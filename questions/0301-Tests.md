# Вопрос

## Темы, теги

Design, code quality, tests

## Текст вопроса

Что лишнее в AAA-шаблоне модульных тестов?

## Варианты ответа

### Верный ответ(ы)

* Присваивание (Assign)

### Другие ответ(ы)

* Данные (Arrange)
* Действие (Act)
* Проверка (Assert)

## Объяснение ответа

ААА-шаблон состоит это Arrange-Act-Assert. Подробнее в [wiki](https://technical-excellence.ru/wiki/UnitTest).
